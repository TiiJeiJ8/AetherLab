### üìÅ Data Handling
- **Multi-format Support**: CSV, Excel (.xlsx, .xls), JSON
- **Upload Methods**: Drag-and-drop & file picker

### üé® Customization
- **Styling**: Color themes (Viridis, Plasma), custom palettes, background colors

- **Annotations**: Mean/median lines, dynamic tooltips, export to PNG

- **Auto Analysis**:

- **Interactive Tools**: Zoom, pan, live rendering toggle

## üöÄ Quick Start

### Requirements
- Python 3.7+
- pip package manager
- Virtual environment (recommended)

### Installation
```bash
# Clone repository
git clone https://github.com/TiiJeiJ8/Fuck-Charts.git
cd Fuck-Charts

# Create virtual environment
python3 -m venv venv
.\venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/macOS

# Install dependencies
pip install -r requirements.txt

# Launch application
python app.py
```
Access at: `http://127.0.0.1:8050/`

## üìñ Usage Guide

1. **Upload Data**
   - Navigate to Preprocessing/Visualization page
   - Supported formats: CSV, Excel, JSON

2. **Create Visualizations**
   - Select chart type from sidebar
   - Configure axes, colors, and styling
   - Toggle live rendering or click "Render Chart"

3. **Data Analysis**
   - **Overview Tab**: Basic dataset stats
   - **Analysis Tab**:
     - Missing value heatmap
     - Descriptive statistics
     - Automatic feature insights

## ‚ö†Ô∏è Known Issues & Tips
Usage Instructions:

Access the application URL (typically http://127.0.0.1:8050/).

Navigate to either the Data Preprocessing or Data Visualization page.

Upload your CSV, Excel, or JSON file in the Data Upload section of the sidebar.

Create Visualizations:

Select your desired Chart Type

Assign appropriate columns to X/Y axes, color groups, size mapping, etc. based on chart requirements

Use the Chart Customization panel below the chart to:

Modify titles, colors, labels

Adjust visual properties (marker opacity, line styles, etc.)

Charts auto-update with Live Rendering enabled, or click Render Chart Manually

Important Notes:

If chart titles/backgrounds don't update immediately: Use Plotly's native controls (top-right corner) ‚Üí Perform any zoom action ‚Üí Click "Auto Scale"

Median/Mean lines won't auto-generate for small datasets

Map tile loading issues:

Check internet connection/VPN status

May indicate server-side access restrictions

Chart residue when switching types (common with maps): Refresh page with F5

Detailed instructions available in GuideLine.pdf (To be written)

Perform Preprocessing:

After upload, review the Data Overview tab for:

Basic dataset statistics

Missing value visualization

Switch to Analysis Report tab for:

Detailed statistical breakdowns

Feature distributions

Advanced insights
